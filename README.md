# Prédiction des Performances et de l'Assiduité des Employés

## Description
Cette application prédit les performances et l'assiduité des employés basés sur des données telles que les heures travaillées, les heures de formation, la performance précédente, l'ancienneté et le genre.

## Prérequis
- Python 3.x
- Les dépendances sont listées dans `requirements.txt`
